# ChangeSpan Support

## Product and engineering support

Use repository issues for reproducible bugs, approved feature requests, documentation problems, and engineering tasks.

For account, organization, licensing, customer-project, or confidential support, contact `eaguiar@change-span.com`.

## Before requesting help

Include the affected product, platform version, repository commit or release, operating system, reproduction steps, expected result, actual result, and sanitized logs.

Security vulnerabilities must follow `SECURITY.md` and must not be posted publicly.
