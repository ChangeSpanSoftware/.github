# ChangeSpan

**Evidence-backed project intelligence for connected project controls.**

ChangeSpan connects schedule, cost, RFIs, specifications, submittals, documents, field updates, decisions, and audit history into one traceable project model.

## Platform

- **ChangeSpan Desktop** — offline-capable authoring, CPM, cost, field capture, and project controls workspaces.
- **ChangeSpan API** — authoritative objects, relationships, events, permissions, and sync services.
- **ChangeSpan Portal** — review, collaboration, approvals, evidence, and external project access.
- **Project Intelligence Engine (PIE)** — grounded explanations, impact analysis, next-action recommendations, and learning signals.
- **ChangeSpan Shared** — canonical contracts, schemas, fixtures, and compatibility rules.

## Engineering principles

- Evidence before recommendation.
- Relationships before isolated records.
- Human approval for authoritative changes.
- Offline-capable, server-confirmed workflows.
- Explainable AI with visible assumptions and confidence.
- Stable contracts across Desktop, API, Portal, and PIE.

## Current program

ChangeSpan is progressing through the Platform 2.0 and ChangeSpan 1.0 engineering program. Core work is currently focused on canonical objects, the Universal Object Graph, project events, search, auditability, recovery, and Project Intelligence Engine foundations.
